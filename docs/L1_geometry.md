# L1 折射几何处方

基于 Robinson (2022) *Lunar Eclipse Phenomena*, arXiv:2112.08966，经物理核对。

## 核心：擦边高度 → 本影内位置

阳光掠过地球大气 limb 被折射 α(h) 弯入本影。折射角正比于切点局地密度：

```
α(h) = α0 · exp(-h/H),   α0 = 0.0204 rad (70 arcmin),   H = 8 km
```

- **α0=70 arcmin 已含切点两侧对称双段**（日-月擦边光线穿大气路程是地平折射 35' 的两倍），不再 ×2。
- α(h) 消色差——颜色梯度全部来自波长相关的**消光**，折射只负责把不同 h 的光分拣到不同本影半径。

落点（带符号，距本影中心）：
```
r(h) = (R⊕ + h) − α(h)·d_moon
```
负值=越过中心落对侧（极低 h 强折射光线）。归一化 `r_norm = |r|/R_umbra`。

本影半径（会聚影锥）：`R_umbra = R⊕ − (R_sun−R⊕)·d_moon/d_sun ≈ 4601 km = 41.2 arcmin`。

## 关键含义：绿松石带是"窄壳"

**映射方向**：低 h → 强折射 → 落本影深处（r_norm→0，红核）；高 h → 弱折射 → 落本影外缘（r_norm→1，青绿/白）。

**绿松石带（h~15-45km）全部挤在 r_norm∈[0.9,1.0] 的薄壳里**——这就是它"窄、marginal、要仔细看"的几何根源。后果：
- **必须以 h 为内部采样变量**（在 h=10-50km 密采），不能按 r_norm 均匀采（会采空）。
- 画图横轴用**角距 arcmin**比 r_norm 更能展开这条带（h=18→49.8', h=50→57.3'，留有展开余地）。

## 聚焦因子（亮度，不碰色相）

```
|dh/dr| = 1/(1 + α(h)·d_moon/H)
```
低 h（强折射）→ 因子小 → 光会聚浓缩 → 本影中心更亮（红核虽消光重却最亮的几何原因）。
乘到亮度 Y 上；色相不受影响。r→0 处需软下限正则化（太阳非点源）。

## air mass 近似

直线切向视线 + ×2 对称双段，对绿松石带（高 h、弱折射）几乎精确；对红核（低 h）低估 slant column <15% 且不改色相。v1 安全。真要精化应改沿折射光线积分（解 Bouguer ODE，同时给 α 和 slant column）。
