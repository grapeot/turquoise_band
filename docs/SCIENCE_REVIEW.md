# Science Review 摘要（多 agent 调研 + 对抗验证）

原始结论见 `science_review_raw.json`。这里是可读摘要，聚焦"要重现绿松石带，物理上必须做对什么"。

## 一、要重现的物理现象

绿松石带是月全食时**本影内边缘**（最靠近半影/亮区那一侧）的一条**窄**青绿/teal 带，夹在铜红色核心和外缘之间。最佳观测时机是全食的头几分钟和最后几分钟，是个 marginal 的肉眼目标。

颜色随**光线擦过地球大气的擦边高度（tangent height）**变化：
- 低擦边高度（对流层 0-12km）的光被瑞利散射严重红化 → 落到本影中心 → 红核
- 高擦边高度（平流层 ~20-40km）的光穿过臭氧层，Chappuis 带吃掉橙红 → 落到本影边缘 → 青绿带

## 二、要用的物理模型

把本影照明建模成对折射太阳光线的求和，按擦边高度 h 参数化，每条光线映射到本影内一个径向位置。每条光线的透射谱：

```
I(λ,h) = I_sun(λ) · exp(-τ_total(λ,h))
τ_total = τ_Rayleigh + τ_O3(Chappuis) + τ_aerosol + τ_cloud
```

沿切向（limb）长程路径积分。各项主导区：
- **瑞利 ∝ λ⁻⁴**：低 h（对流层）主导 → 红
- **臭氧 Chappuis**（400-650nm，峰 ~600nm，峰值截面 ~5.2e-21 cm²）：高 h（平流层）主导 → 青绿

### 安全的近似
- 单次散射 + 消光-only（瑞利当消光项）：对"颜色 vs 径向位置"的定性形状足够
- 球对称大气、指数空气密度、标准臭氧廓线（~300 DU）
- 5772K 黑体太阳谱：对色相趋势够用

### 危险的近似（会做错）
- **省掉折射**：光根本进不了本影，h↔径向映射也错。光是被双 limb（两侧大气）折射进来的，总偏转 ~1-1.2°，slant air mass 比单边再翻倍
- **省掉臭氧**：直接丢掉整个绿松石现象。纯瑞利只会单调红化，给不出蓝边
- **臭氧当均匀板而非分层**：带的边缘定位依赖臭氧是平流层薄层
- **臭氧柱量不乘 air-mass 增强**：limb 路径 τ_O3 要 ×20-40 才到 0.8-1.6，才够把橙红压成青绿
- **消光-only 用于绝对亮度**：深本影的多次散射/大气聚焦会被低估（亮度量级需要完整辐射传输，颜色形状不需要）
- **折射当波长相关**：折射近乎消色差，颜色梯度完全来自波长相关的**消光**，不是色散弯曲

## 三、颜色科学管线（关键陷阱）

谱 → ∫I·CMF dλ → XYZ → sRGB；色相用 CIELAB hue angle。**两个致命陷阱**：

1. **归一化常数 k 必须从未衰减入射日光算一次，全程复用**。不能 per-height 归一，否则本影变暗的亮度阶梯就被抹平了——而亮度对比正是这个现象的本质。Y(h) 应该跨好几个数量级，画对数轴。
2. **提取色相前不能 per-channel 截断 sRGB**。高饱和的青绿/深红在 linear sRGB 经常越界，逐通道 clip 会扭曲色相。色相/Lab 分析要在未截断的线性值上做，gamut 映射只在最后显示时朝白点去饱和。

预期色相轨迹：深本影血红（Lab hue ~25-50°）→ 绿松石带（hue ~190-220°）→ 趋白（低饱和）。
