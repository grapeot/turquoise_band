# 物理方法与踩坑记录

这份文档系统记录绿松石带项目"物理怎么做的"和"踩了哪些坑、怎么解的"。给后来者（包括未来的自己）一个不必重走弯路的地图。配套：`PRD.md`（目标）、`RFC.md`（管线设计）、`L1_geometry.md`（折射几何）、`SCIENCE_REVIEW.md`（文献调研）、`LOG.md`（时间线日志）。

## 一、现象与核心物理判断

月全食时月面不是全黑，而是被折射进地球本影的阳光照亮。沿月面径向，从本影中心到边缘对应阳光擦过地球大气的不同"擦边高度"，颜色因此变化。三个区，各有一个 dominant 物理量：

- **本影深处（红核）** — 阳光擦过稠密低层大气，瑞利散射 ∝λ⁻⁴ 把蓝光几乎散尽，只剩红光穿透 → 铜红（血月）。dominant：瑞利散射。
- **绿松石带（窄）** — 阳光擦过平流层，瑞利对蓝端的衰减已弱，但臭氧 Chappuis 吸收带（~500-650nm，峰 575/603nm）选择性吃掉橙红 → 残留蓝绿。dominant：臭氧 Chappuis 吸收。
- **边缘（白）** — 擦边高度高、光程薄，接近未经强选择性衰减的原始日光 → 白。dominant：低光学厚度。

绿松石带是**臭氧吸收主导、瑞利退居次要**的高度窗口的产物。这是整个项目的物理主线。

## 二、计算管线（第一性）

```
擦边高度 h
  │  几何：切向穿过球壳大气，沿视线采样 (高度, 弧长)
  ▼
柱密度 N_air(h), N_O3(h)   ← 对 AFGL 实测大气廓线沿视线积分（对称双段 ×2）
  │
  ▼
光学厚度 τ(λ,h) = N_air·σ_rayleigh(λ) + N_O3·σ_O3(λ)
  │   σ_rayleigh: 解析(含折射率色散)；σ_O3: Serdyuchenko 2014 实测截面
  ▼
透射率 T(λ,h) = exp(-τ)        ← Beer-Lambert，无自由参数
  │
  ▼
出射谱 I(λ,h) = I_sun(λ)·T(λ,h)   ← I_sun: 5772K 黑体(对色相趋势够用)
  │
  ▼
颜色 XYZ = ∫I·CMF dλ → sRGB / 色相 / 亮度   ← CIE 1931 2° CMF
  │
  ▼
折射几何：h → 月面距本影中心角距/径向位置
  │
  ▼
色相曲线(L1) / 月盘渲染(L2)
```

数据全部权威实测：臭氧截面 Serdyuchenko 2014（603nm 峰 5.15e-21，与文献吻合1%）、大气廓线 AFGL US Standard、CIE CMF（CVRL）。来源见 `data/raw/SOURCES.md`。

## 二·五、统一方法论：一个多维积分的 ray tracing

整个项目本质上是**一个 ray tracing**，在若干维度上积分。把它讲清楚，既是方法说明，也是未来做归因分析（ablation study）的维度清单——每个维度都能单独开关，看它对绿松石带的贡献。

**积分维度分两类：空间有限元 × 光谱。**

**（A）三个空间有限元维度**——每个都是"撒很多 ray / 像素再积分"：

1. **太阳圆盘**（角半径 16'）。太阳不是点源，圆盘上每个子点是一束方向略不同的平行光。这是 2026-05-31 才补上的维度（之前用点源近似）。**影响巨大**：太阳全宽 32' 比整条绿松石带角宽（~6'）还宽 5 倍，相当于一个比信号还宽的卷积核，把窄青谷糊平——点源版算出浓青 R/B=0.41@53'、锐边 1.2'，圆盘版才是真实的浅青 R/B=0.71@40'、软边 3.5'（金标准 `brute_ray_trace.py` 三方向收敛、退化点源精确复现验证）。
2. **地球大气高度** h（擦边高度）。这是物理主轴。**h 同时决定两件事**：(a) 折射偏转角 α(h)=0.0204·exp(-h/8)——决定这条 ray 弯多少、落到月面哪——(b) 沿视线的大气柱密度 N_air(h)、N_O3(h)——决定吸收/散射多少、什么颜色。低 h（稠密低层）强折射+强消光→红核；中 h（平流层）→臭氧主导→青；高 h（稀薄）→弱折射弱消光→白。
3. **月面像素**。每个月面点是一个落点，收到的是"所有最终折射落到它这里的 (太阳子点 × 擦边高度) 组合"的光谱叠加。视频里再加月面纹理（反照率）调制亮度。

**（B）三条光谱**——都是按波长 λ 的函数，逐 ray 相乘：

1. **太阳光谱** `I_sun(λ)`——入射源（SAO2010 实测 / 5772K 黑体）。
2. **瑞利散射谱** `σ_Rayleigh(λ)∝λ⁻⁴`——蓝端损耗远大于红端，血月红的成因。机制是散射（光子被散射出视线，对透射光等效为损失），不是吸收。
3. **臭氧吸收谱** `σ_O3(λ)`——Chappuis 带（~500-650nm），选择性吃掉橙红，绿松石带的成因。机制是真·吸收。

合成：`T(λ,h)=exp(−(N_air·σ_Rayleigh + N_O3·σ_O3))`，出射谱 `I(λ,h)=I_sun(λ)·T(λ,h)`，积成 XYZ。

**（C）第四维：时间**——视频里月心距 D 随帧变化（月盘扫过本影），研究图固定单帧。

**已知漏掉/简化的维度（ablation 时要标明）：**

- **折射色散**：α(h) 当成消色差（所有波长同一偏转角）。真实大气有色散——蓝光折射比红光强（像棱镜），不同颜色落点微微错开。当前模型没有这一维，可能对绿松石带的色彩边缘有细微影响。**这是最值得补的漏项。**
- **单次散射**：瑞利散射的光当"散射出视线即损失"，不追被散射进视线的二次光。文献证实对绿松石带颜色影响不大，未做。
- **气溶胶/云**：未建模（真实月食的红核深浅很大程度由当时平流层气溶胶决定，但对"是否有绿松石带"这个定性结论无关）。

**为什么这个框架适合 ablation**：每个维度独立可关——太阳圆盘 on/off、瑞利 on/off、臭氧 on/off、色散 on/off、太阳谱黑体/实测——逐一开关就能量化每个机制对绿松石带（颜色 R/B、位置、亮度悬崖）的贡献。这正是"逐因素叠加 demo"的物理基础。

## 三、踩过的坑（按时间，附根因和解法）

### 坑1：颜色归一化做成 per-height（L0 v0）
**症状**：本影变暗的亮度阶梯被抹平，每个高度都"满亮度"。
**根因**：每个擦边高度各自归一化颜色，丢掉了绝对亮度信息。
**解法**：归一化常数 k 从**未衰减入射日光**算一次，全程复用，让白点 Y=1。这样 Y(h) 自动携带真实消光导致的亮度暴跌。science review 独立地把这列为标准做法、把 per-height 归一列为致命错误。
**教训**：归一化不是为了好看，是为了物理正确——错误的归一化会**制造**假象，正确的归一化**避免**假象。

### 坑2：臭氧 Chappuis 用窄高斯近似（L0 v0）
**症状**：绿松石带偏蓝紫，红端透射过高。
**根因**：双高斯近似把 Chappuis 带做得太窄（集中 575-602nm），对红端吸收不足。
**解法**：换 Serdyuchenko 2014 实测截面。真实带形更宽、红端衰减更平缓。

### 坑3：聚焦因子方向写反（L1）
**症状**：本影中心被压暗到纯黑，红核消失。
**根因**：初版用 `1/(1+α·d/H)`，漏了环带几何的 `/r` 项，方向反了。
**解法**：回到环带能量守恒重推 `B ∝ b·|dh/dr|/r`。低 h 光线会聚到近中心(小r)→中心更亮（这才是红核虽消光重却最亮的几何原因）。
**教训**：发现红核太暗时**没有**去加个亮度系数凑（那是 reactive），而是回到第一性发现漏项。

### 坑4（最深）：用一条公式贯穿红核到绿松石带（L1）
**症状**：绿松石带(h~20-40km)被算到 50-57'，落在本影几何边界(41')外 10-16'，与文献"紧贴边界、外移仅0.78-1.9'"矛盾 5-10×。
**根因（第一性）**：公式 `r(h)=(R⊕+h)−α·d_moon` 是太阳**贴影锥轴那一侧 limb** 发出、向轴弯的光线落点——它描述**红核**对（低 h 强折射→落本影深处）。但绿松石带在月盘**外缘**，照亮它的光来自太阳**对侧 limb**，穿过地球另一侧的平流层臭氧层。把贴轴公式外推到高 h，光线滑出本影落到半影侧，这才是错位来源。
**解法**：绿松石带用镜像的对侧-limb 公式 `r_opp(h)=R_umbra−(α·d_moon−h)`，归一化基准用 R_umbra 而非 R⊕。修正后绿松石带落 34-41'，紧贴边界，与 Mallama/Robinson 自洽。
**教训**：红核和绿松石带来自太阳圆盘的两个不同 limb，是镜像关系，不能用一套映射贯穿。这是个"用对了物理才出来"的修正，不是调参。

### 坑5：拿 CIELAB hue 角当色相判据，被它误导（L1）
**症状**：绿松石带 CIELAB hue=267°（看着像饱和蓝），怀疑物理偏蓝太多，差点去改物理凑 teal。
**根因**：CIELAB 的 b*=−23.8（强偏蓝）把视觉上**温和**的偏蓝放大了（a*≈0，其实不偏绿）；加上白点用 5772K 黑体偏暖，相对它做 Lab 会把中性谱推得更蓝。
**真相**：出射谱蓝/绿黄=1.44 是 Chappuis 的真实物理；sRGB 红蓝比 0.88 **正好落在文献"微偏蓝"区间 0.8-1.0 内**。模型是对的，是**度量误导了判断**。
**解法**：改用 sRGB 红蓝比当判据（比 CIELAB hue 角诚实），不碰任何物理参数。
**教训**：色相偏差先怀疑**度量**，再怀疑物理。绝不为了凑一个漂亮的色相角去改物理。

## 四、安全 vs 危险的近似（来自 science review）

**安全（对颜色趋势足够）**：单次散射 + 消光-only（瑞利当消光项）；5772K 黑体太阳谱；球对称大气；直线切向视线算柱密度（对绿松石带的高擦边高度几乎精确）。

**危险（会做错）**：
- 省掉折射 → 光进不了本影，h↔位置映射也错。
- 省掉臭氧 → 直接丢掉整个绿松石现象（纯瑞利只会单调红化）。
- 臭氧当均匀板而非平流层薄层 → 带的边缘定位错。
- 臭氧柱量不乘 limb air-mass 增强 → 吸收不够，绿松石出不来。
- 颜色 per-height 归一化 → 抹平亮度阶梯（坑1）。
- 提取色相前 per-channel 截断 sRGB → 扭曲高饱和色的色相。

## 五、贯穿全程的方法论原则

1. **每一步都有能直接看的标量/曲线**：透射谱→XYZ→色相/亮度→曲线。先确认形状对，再提分辨率、换真实数据、上渲染。
2. **每替换一个近似为真实数据，对比曲线确认没回归**。
3. **物理正确性 > 好看**。绝不为凑结果引入假过程或假归一化。出现"看着不对"时，先回第一性查物理或查度量，不做 reactive 的事后校正。
4. **凑数的诱惑出现时记下来**（如坑3/4/5），它们往往指向更深的物理理解。

## 六、可观测交叉验证锚点（文献）

- 折射：地表掠射偏转 α(0)≈70 arcmin；月距偏移≈7800km（填满本影）。Robinson 2022 (arXiv:2112.08966) Table 3.1。
- 本影角半径 ≈41.2'；绿松石带紧贴边界、外移 0.78'(观测)~1.9'(模型)。Mallama 2021。
- 臭氧 Chappuis：峰 575/603nm，截面 4.83/5.23e-21 cm²。
- 绿松石带真实色相：定量实测红蓝比 0.8-1.0（微偏蓝，肉眼难辨，光谱明显）。
- 颜色+物理金标准：Gedzelman & Vollmer 2008 (Appl. Opt. 47:H149)；Garcia-Muñoz & Pallé 2011。
- 本影径向亮度剖面 golden：Mallama 2021 Table B-1。
