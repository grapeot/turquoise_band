# 项目日志

## L0 v0（解析近似闭环）— 已跑通，趋势部分对

管线全链路打通：大气廓线 → limb 柱密度 → τ(λ,h) → 透射率 → 出射谱 → XYZ → 色相/亮度曲线。
六个底层模块各自自查通过（瑞利截面量级对、几何对、颜色管线对、臭氧层峰值在平流层）。

### 已捕捉到的正确物理
- 低擦边高度蓝光被瑞利散射几乎全灭、红光透过（血月成因）：10km 处 T@450nm=0.014 vs T@680nm=0.31。
- 中等高度（~25km）出现臭氧 Chappuis 吸收谷：T@600nm 明显低于两侧。
- 亮度随擦边高度单调上升（100%），符合"越靠边越亮"。

### v0 近似的两个偏差（待真实数据校核）
1. **本影深处偏黄橙而非深红**（色相角 ~55-78°）。怀疑：最深擦边高度的 limb 光程/瑞利消光还不够极端；可能需要更低的 h_min 或更长的 s_max，也可能黑体日光近似偏暖。
2. **绿松石带偏蓝紫而非青绿**（21-28km 色相角 245-259°，应 ~180°）。根因诊断：v0 的 Chappuis 双高斯近似带形太窄（集中 575-602nm），对红端(660-700nm)吸收不足，导致红光大量透过，蓝+红同强 → 偏紫蓝。
   - **预期修复方向**：真实 Serdyuchenko 2014 截面带更宽、红端衰减更平缓；真实 limb 臭氧柱量（~数百 DU × 长程air mass）远大于近似，会把橙红压得更深 → 谱真正偏青绿。
   - 这是"用真实数据替换近似"应当修正的典型回归，待数据接入后对比 v0/v1 曲线确认。

### 结论
管线正确，物理骨架对。绿松石带的色相精度依赖真实臭氧截面 + 真实臭氧柱密度，这正是下一步接入权威数据要解决的。

## L0 v1（真实数据接入）— 闭环自查全部通过 ✓

接入权威数据并按 science review 修正后重跑：
- 臭氧截面：Serdyuchenko 2014（233K 平流层列），603nm 峰值 5.15e-21 cm²（文献 5.2e-21）
- 大气廓线：AFGL US Standard（空气数密度 + 臭氧 VMR→数密度，峰在 22km）
- 颜色归一化修正：k 从未衰减入射日光算一次全程复用，色相用同一固定白点（修了 v0 的 per-curve 归一错误）

### v1 自查结果（全过）
- 暖色红区：2–14km，亮度 Y=4.2e-3 ~ 1.1e-1（极暗血月深处）
- 绿松石冷色带：16–36km
- 趋白区：36–70km
- 亮度单调升 100%，动态范围 240 倍（边缘/深处）
- 六项断言全过：红区/青绿带/带高于红区/趋白区/单调/亮度暴跌

合成色带（outputs/L0_colorband.png）肉眼可见：深暗红 → 青绿 teal 带 → 白。**核心物理（红→青→白 + 本影变暗阶梯）重现。**

### 仍存的二阶偏差（留待 L1）
- 绿松石带色相偏蓝青（曲线 250-267°，理想 ~190-220°）。即蓝端透射偏多/橙红压得还不够狠。
- 可能原因：(a) 几何用直线视线 + 线性 h↔r 映射，未含真实折射；science review 强调需双 limb 折射几何 + air-mass 增强的臭氧柱量；(b) 太阳谱用黑体（review 说对趋势够用，但精修可换 ASTM/SAO2010）；(c) 缺气溶胶/对流层云的低空遮挡。
- 这些都不影响 L0 的定性结论，属于 L1 定量校核的范畴。
