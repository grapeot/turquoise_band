# 项目日志

## L0 v0（解析近似闭环）— 已跑通，趋势部分对

管线全链路打通：大气廓线 → limb 柱密度 → τ(λ,h) → 透射率 → 出射谱 → XYZ → 色相/亮度曲线。
六个底层模块各自自查通过（瑞利截面量级对、几何对、颜色管线对、臭氧层峰值在平流层）。

### 已捕捉到的正确物理
- 低擦边高度蓝光被瑞利散射几乎全灭、红光透过（血月成因）：10km 处 T@450nm=0.014 vs T@680nm=0.31。
- 中等高度（~25km）出现臭氧 Chappuis 吸收谷：T@600nm 明显低于两侧。
- 亮度随擦边高度单调上升（100%），符合"越靠边越亮"。

### v0 近似的两个偏差（待真实数据校核）
1. **本影深处偏黄橙而非深红**（色相角 ~55-78°）。怀疑：最深擦边高度的 limb 光程/瑞利消光还不够极端；可能需要更低的 h_min 或更长的 s_max，也可能黑体日光近似偏暖。
2. **绿松石带偏蓝紫而非青绿**（21-28km 色相角 245-259°，应 ~180°）。根因诊断：v0 的 Chappuis 双高斯近似带形太窄（集中 575-602nm），对红端(660-700nm)吸收不足，导致红光大量透过，蓝+红同强 → 偏紫蓝。
   - **预期修复方向**：真实 Serdyuchenko 2014 截面带更宽、红端衰减更平缓；真实 limb 臭氧柱量（~数百 DU × 长程air mass）远大于近似，会把橙红压得更深 → 谱真正偏青绿。
   - 这是"用真实数据替换近似"应当修正的典型回归，待数据接入后对比 v0/v1 曲线确认。

### 结论
管线正确，物理骨架对。绿松石带的色相精度依赖真实臭氧截面 + 真实臭氧柱密度，这正是下一步接入权威数据要解决的。**stop 点：等真实数据接入，重跑曲线，对比 v0→v1 的色相是否从偏蓝紫收敛到青绿。**
